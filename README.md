
## EXE Build

One-file build files are prepared for this project:

- `build-exe.bat`
- `sg-editor.spec`
- `requirements-build.txt`

Build command:

```bat
build-exe.bat
```

The build embeds:

- `assets/`
- `discordrpc/`
- `editor_layout.json`
- `app_settings.json`

The resulting file is created at `dist\SGMEditor.exe`.
