# LMR Mod Editor

Простой настольный MVP-редактор модов для **Love Money Rock'n'Roll** на **Ren'Py** без внешних зависимостей.

## Что уже умеет

- создавать каркас нового мода;
- открывать папку проекта;
- показывать дерево файлов;
- редактировать `.rpy`, `.rpym`, `.json`, `.txt`, `.md` и другие текстовые файлы;
- искать текст по всему проекту;
- вставлять быстрые Ren'Py-сниппеты;
- строить outline по `label`, `screen`, `menu`;
- экспортировать проект в `.zip`.

## Запуск

```powershell
python app.py
```

## Структура нового проекта

После создания редактор генерирует:

- `mod_info.json` - базовая информация о моде;
- `game/<mod_id>/script.rpy` - точка входа;
- `images/` - графика;
- `audio/` - музыка и звуки;
- `docs/` - заметки.

## Идеи для следующего шага

- подсветка синтаксиса Ren'Py;
- визуальный редактор персонажей, сцен и спрайтов;
- менеджер переводов;
- упаковка мода в формат, который использует ваша конкретная сборка игры;
- валидация ссылок на изображения, аудио и `label`.

## Важное допущение

Этот MVP работает как редактор **распакованного проекта мода**. Если у вашей версии Love Money Rock'n'Roll есть строгий формат загрузки модов, его можно добавить следующим этапом.
